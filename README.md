# 🏠 My Home Lab chart repository

## How to use?

1. Agregar el repositorio
```bash
helm repo add cubancodelab https://helms.cubancodelab.net
```

2. Actualizar
```bash
helm repo update
```

3. Instalar budget-api
```bash
helm install zero-api homelab/zero-api
```
## Available Charts

- **zero-api** - API to connect to Actual Budget (Integrate with ZeroTap)
